## Test Demo API
### Create User

- *Method:* `POST`
- *URL:* `https://my-json-server.typicode.com/javoweb/typicode-demo/users/`
- *Data:* `{"firstName": "<first name>", "lastName": "<last Name>"}`

### Get Users List

- *Method:* `GET`
- *URL:* `https://my-json-server.typicode.com/javoweb/typicode-demo/users/`

### Get single User

- *Method:* `GET`
- *URL:* `https://my-json-server.typicode.com/javoweb/typicode-demo/users/:id`

### Update User

- *Method:* `PUT`
- *URL:* `https://my-json-server.typicode.com/javoweb/typicode-demo/users/:id`
- *Data:* `{"firstName": "<first name>", "lastName": "<last Name>"}`

### Delete User

- *Method:* `DELETE`
- *URL:* `https://my-json-server.typicode.com/javoweb/typicode-demo/users/:id`
